# DESKRIPSI APLIKASI


Aplikasi Pencarian Cepat Kata Kunci pada Dokumen Digital ini merupakan aplikasi berbasis web yang digunakan untuk membantu pengguna dalam mencari kata kunci tertentu pada kumpulan dokumen teks secara cepat dan efisien. Aplikasi ini menerapkan metode string matching dengan menggunakan algoritma Knuth–Morris–Pratt (KMP).
Pada aplikasi ini, pengguna dapat memasukkan kata kunci melalui kolom pencarian yang tersedia di halaman utama. Setelah kata kunci dimasukkan, sistem akan melakukan pencarian pada seluruh dokumen teks yang terdapat dalam folder dataset. Proses pencarian dilakukan dengan algoritma KMP sehingga pencocokan kata dapat dilakukan lebih cepat tanpa mengulang pencarian dari awal.
Hasil pencarian akan ditampilkan dalam bentuk daftar dokumen yang mengandung kata kunci tersebut. Setiap hasil menampilkan nama file dan cuplikan teks yang mengandung kata kunci, di mana kata kunci akan ditandai dengan highlight agar mudah dikenali. Pengguna juga dapat mengklik salah satu hasil untuk melihat isi dokumen secara lengkap, dan kata kunci tetap ditampilkan dengan highlight di dalam dokumen.
Aplikasi ini dibangun menggunakan PHP sebagai bahasa pemrograman utama, serta HTML dan CSS untuk tampilan antarmuka. Dengan tampilan yang sederhana dan mudah digunakan, aplikasi ini diharapkan dapat membantu pengguna dalam melakukan pencarian informasi pada dokumen digital, khususnya dalam jumlah data yang cukup banyak.
